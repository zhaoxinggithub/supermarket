# java-cs-shop
一个基于CS架构、Socket、多线程，用swing写的超市收银管理系统。

### 实现功能

* 登陆、注册
* 扫描商品
* 结账
* 找零
* 商品的增、删、改、查
* 用户的增、删、改、查
